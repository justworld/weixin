# weixin
微信网页版<br>
聊天通过websocket实现，websocket项目<br>
后台有django和flask两个版本，django_api、flask_api<br>
vue.js前端<br><br><br>
# 运行
前台进入web文件夹，运行npm install安装依赖，npm run dev运行<br>
后台分别根据django、flask项目运行
